# Cotton Eye Pomodoro

A fun Pomodoro timer app that plays "Cotton Eye Joe" song when you get distracted.

## Quickstart

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sanderjoon/cotton-eye-pomodoro.git
   cd cotton-eye-pomodoro
   ```

2. **Run the app:**
   ```bash
   python cotton-eye-pomodoro.py
   ```

3. **Configure settings:**
   Edit `pomodoro_settings.json` to customize your work/break intervals and other preferences.

## Features

- Pomodoro timer with customizable work and break intervals
- Motivational quotes played during breaks
- "Cotton Eye Joe" plays if you get distracted
- Audio notifications for session changes

## Requirements

- Python 3.x
- Audio playback capability on your system
